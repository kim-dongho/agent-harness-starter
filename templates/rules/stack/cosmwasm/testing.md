# CosmWasm — 테스트
- `cw-multi-test`로 여러 컨트랙트 간 상호작용을 통합 테스트한다.
- `mock_dependencies()`와 `mock_env()`로 유닛 테스트 환경을 구성한다.
- 권한 없는 호출, 잔액 부족 등 실패 케이스를 반드시 테스트한다.
- IBC 시나리오 테스트를 포함한다.
