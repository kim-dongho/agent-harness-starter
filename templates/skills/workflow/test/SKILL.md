### 2. Self-healing Loop

실패 시 자가 수정 루프를 따른다:

1. 테스트 실행 → 실패 로그 확인
2. 에러 메시지를 분석하여 원인 파악
3. 코드 수정
4. 테스트 재실행
5. **최대 3회 반복** — 3회 실패 시 사용자에게 보고하고 중단

```
retry 1/3: ❌ expect(result).toBe(true) → 원인 분석 → 수정 → 재시도
retry 2/3: ✅ 통과
```
